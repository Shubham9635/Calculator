# 🧮 Calculator Web App

A clean and responsive **Calculator** built using **HTML, CSS, and JavaScript**.
This project demonstrates core front-end skills such as DOM manipulation, event handling, and responsive UI design.

## ✨ Features

* ➕ Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
* 🎨 Modern and user-friendly interface
* ⚡ Fast and lightweight
* 🔄 Clear/Delete functionality
* 📱 Responsive design (works on mobile & desktop)
* ⌨️ Button-based input handling

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & layout
* **JavaScript (Vanilla JS)** – Calculator logic

## 🧩 How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/calculator.git
   ```

2. Open the project folder.

3. Run `index.html` in your browser.

✅ Done!

---

## 🌐 How to Deploy (GitHub Pages)

1. Go to **Settings** of this repository
2. Open **Pages** section
3. Select branch → `main`
4. Save and get your live link 🎉
